前端代码：SpringWeb/src/main/resources/static
后台代码：SpringWeb/src/main/java/com/example/demo

static包 -- 前端css、js、html文件存放处

controller包 -- 控制类包，接收前端传来的信息，控制程序流程

service包 -- 程序逻辑设计包，完成各项增删改查的功能

repository -- 数据库操作包

dao -- 实体类包
